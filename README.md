# 커머스 과제


## 클래스
- CommerceSystem : 비즈니스 로직 처리
- Admin : 관리자 로직 처리(상품 관리)
- Category : 상품 카테고리 관리
- Database : 상품 및 리스트 정보 저장
- Product : 상품 정보(상품 이름, 가격, 설명, 수량 등) 저장

### 기능
1. 상품 등록
    - 상품 리스트에 상품 등록 후 List<Category> 리스트에 저장
   
2. 리스트 및 상품 조회
    - getter 및 setter를 통해 리스트 및 상품에 접근 및 저장

3. 장바구니에 추가
   - 상품 정보 입력 후 리스트에 추가
